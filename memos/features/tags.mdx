## 什么是标签

在 memo 内容里输入 `#标签名`，这条 memo 就自动带上了该标签。标签名以空格、换行或文末结束。

例如：

```
今天读了一篇很有意思的论文 #读书 #灵感
```

这条 memo 会同时出现在 `#读书` 和 `#灵感` 两个标签下。

## 侧边栏的标签列表

侧边栏「Tags」区域会列出你用过的所有标签，以及每个标签下的 memo 数量。点击任意标签即可过滤主列表，只显示包含该标签的 memo。

再次点击同一标签，或点击列表顶部的「All」，即可清除过滤，回到全部 memo。

## 在 memo 卡片上点击标签

memo 卡片底部会显示该条 memo 的所有标签。点击卡片上的标签与点击侧边栏标签效果相同，会立即过滤到对应标签视图。

## 清除标签过滤

有两种方式清除当前激活的标签过滤：

- 点击侧边栏「Tags」列表顶部的 **All**
- 点击页面标题旁显示的标签 badge 上的 **×**

## 标签的自动维护

标签不需要手动创建或删除。删除一条 memo 后，如果该标签下已没有其他 memo，标签会自动从侧边栏消失。同理，归档的 memo 的标签仍然保留，因为归档的 memo 仍然存在。
