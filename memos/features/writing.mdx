## 编辑器

主页顶部始终显示一个输入框，直接点击即可开始输入。编辑器会在你聚焦时高亮边框，失焦后恢复静默状态。

写完后有两种发送方式：

- 按 **⌘ Enter**（macOS）或 **Ctrl Enter**（Windows / Linux）
- 点击右下角的 **Send** 按钮

内容为空时 Send 按钮为禁用状态，不会误发空 memo。

## Markdown 格式

memo 内容支持以下 Markdown 语法：

| 格式 | 语法 | 示例 |
|---|---|---|
| 粗体 | `**文字**` | **文字** |
| 斜体 | `_文字_` | _文字_ |
| 行内代码 | `` `代码` `` | `代码` |

编辑器工具栏提供对应的快捷按钮：**B**（粗体）、_I_（斜体）、`<>`（代码）。选中文字后点击按钮，会自动在选中内容两侧插入符号。

## 插入标签

工具栏的 **#** 按钮会在当前光标位置插入 `#`，你只需要继续输入标签名即可。标签以空格结尾自动识别。

也可以直接在内容任意位置手动输入 `#标签名`。

## 字符统计

开始输入后，编辑器右下角会显示当前内容的字符数和行数，方便你感知内容体量。

## 编辑已有 memo

悬停任意 memo 卡片，点击右侧出现的铅笔图标，会弹出编辑弹窗。弹窗使用相同的编辑器，同样支持 **⌘ Enter** 快捷键提交。提交后弹窗自动关闭，内容立即更新。

<Note>
  编辑 memo 会记录「edited」标记，显示在卡片右下角时间旁边。
</Note>
